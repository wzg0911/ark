## 技术笔记

### Juejin API 正确格式（2026-06）
- Draft创建需 `edit_type: 10`（markdown）和 `html_content: "deprecated"`
- Publish仅需 `draft_id` + `sync_to_org` + `column_ids` + `theme_ids`
- Tag IDs 必须是字符串数组（不是数字）
- 文章发布后进入审核状态，审核通过后 `/post/` URL 可访问

### dev.to API
- `POST https://dev.to/api/articles` 直接发布
- 需要 Header `api-key` 认证
- 标签会自动创建如果不存在
